\# Parte 1 - Desenvolvimento Front-End



\## 1. Proposta da Aplicação



O projeto "Sistema de Gerenciamento de Corridas" foi desenvolvido para auxiliar organizadores de corridas amadoras no gerenciamento de eventos esportivos e participantes inscritos. A aplicação busca centralizar informações importantes como dados das corridas, categorias e status de inscrição dos atletas.



Os principais usuários da aplicação são organizadores de eventos esportivos e participantes das corridas.



As duas entidades centrais da aplicação são:

\- Corrida

\- Participante
